Chaining with jQuery

Almost all jQuery methods return jQuery objectâwhich allows for chaining.

Instead of performing DOM operations line-by-line, we can chain elements.

Instead of:

  console.log("button clicked!");
});

Why Use On()?

In most cases, click() and on(âclickâ) will both get the job done.
HOWEVER, there is one key difference:

  • click() only adds listeners for existing elements
  • on() will add listeners for all potential future elements
//alerts when ANY button is clicked

  console.log("todo was clicked!");
});

This allows us to leverage event delegation
