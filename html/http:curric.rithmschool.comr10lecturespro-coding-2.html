Types in JavaScript

Primitives

  • Primitive types are also known as scalar or simple types.
  • Primitive types fit into memory easily.
  • JavaScript has six native primitive types
    • null
    • undefined
    • Boolean
    • Number
    • String
    • Symbol

Reference Types

  • A reference type can contain multiple values.
  • Reference types are also known as: complex types or container types.
  • Reference types in JavaScript include:
    • objects
    • arrays (a kind of object)
    • functions (a kind of object)

Assigning a primitive

  • When a primitive type is assigned to another variable, the second variable simply points to the first value.

Assigning a reference

When a reference type is assigned to another variable, the address of that value is what is copied over.

let names = ["Joel", "Alissa", "Michael"];
let namesCopy = names

Comparing two primitives

  • When the comparison operators ( == and ===) are used on primitives, they check the type and value.
let firstName = "Elie"
let firstNameAgain = firstName
firstName === firstNameAgain // true

firstName === "Elie" // true
firstNameAgain === "Elie" // true

Comparing two references

  • When the comparison operators (== and ===) are used on reference types, they check the reference.
  • If the variables contain a reference to the same item, the comparison will result in true.
let nums = [1,2,3,4]
let nums2 = nums
nums === nums2 // true

nums === [1,2,3,4] // false
nums2 === [1,2,3,4] // false

So how do we compare two objects?

The quick way:

let instructorData = {name: "Elie"}
let instructorData2 = {name: "Elie"}

let instructorDataToString = JSON.stringify(instructorData);
let instructorData2ToString = JSON.stringify(instructorData2);
instructorDataToString === instructorData2ToString; // true

Another option would be to recursively loop through the objects and make sure each of the properties/values are the same.
