How to Apply

Cold Application Process

  1. Fill out job applications via the previously-mentioned platforms, or if you have a specific company in mind you can visit their website directly.
  2. Fill out as much info as you can, including optional fields. For instance, if it says cover letter optional, you should definitely write a cover letter.
  3. Add every application to your board in huntr.co using the Chrome extension.
  4. Strive to fill out 3-5 applications daily, ~5 days a week, on average.
